✅ LIKE SYSTEM (Drop-in Code)

Works for Posts, Comments, Replies (just change the path).

Atomic Like Toggle

async function toggleLike(ref, userId) {
  const likeRef = ref.collection("likes").doc(userId);
  const doc = await likeRef.get();

  if (doc.exists) {
    // UNLIKE
    await likeRef.delete();
    await ref.update({
      likeCount: firebase.firestore.FieldValue.increment(-1)
    });
  } else {
    // LIKE
    await likeRef.set({ userId, likedAt: Date.now() });
    await ref.update({
      likeCount: firebase.firestore.FieldValue.increment(1)
    });
  }
}

Usage

Like a post

toggleLike(db.collection("posts").doc(postId), userId);

Like a comment

toggleLike(db.collection("posts").doc(postId)
  .collection("comments").doc(commentId), userId);

Like a reply

toggleLike(db.collection("posts").doc(postId)
  .collection("comments").doc(commentId)
  .collection("replies").doc(replyId), userId);

✅ REAL-TIME COUNTERS (No lag)

This makes sure your feeds load instantly, counts update instantly.

Listen to a Post

db.collection("posts")
  .doc(postId)
  .onSnapshot(doc => {
    const post = doc.data();
    setPost(post);
  });

Listen to Comments

db.collection("posts").doc(postId)
  .collection("comments")
  .orderBy("createdAt", "asc")
  .onSnapshot(snapshot => {
    const comments = snapshot.docs.map(d => ({ id: d.id, ...d.data() }));
    setComments(comments);
  });

Listen to Replies

db.collection("posts").doc(postId)
  .collection("comments").doc(commentId)
  .collection("replies")
  .orderBy("createdAt", "asc")
  .onSnapshot(snapshot => {
    const replies = snapshot.docs.map(d => ({ id: d.id, ...d.data() }));
    setReplies(replies);
  });
